About Me

As a tech and science enthusiast, I have always loved knowing how the universe works and how I can help to improve people's lives. This passion led me to choose a career as a science teacher (physics, mathematics, and robotics). After a few years of teaching, I realised that I had great potential to contribute to the tech industry. Confident that the skills and expertise I developed add really well to the tech career, I decided to migrate to it.

I started my studies in front-end web development with the famous triad: HTML, CSS, and JavaScript. Nowadays, I am fully engaged in the CYF Course, keeping React, Next.js, and TypeScript on my radar.

Well, feel free to check out my portfolio. I'm still in the beginning, but there's already a lot of affection and dedication involved. If you liked what you saw, get in touch. I am always available to help, share, and learn.

Projects Showcase

  • Personal Page

    My personal page built with HTML, CSS, and JavaScript, in the beginning of the CYF course.

    This project was useful to learn and practice the HTML structure, the file organization, and some simple JavaScript functions.

    I could test some techniques to make the hamburger menu with CSS and JavaScript, using 'querySelector' and 'querySelectorAll' to get the HTML tags and manipulate their classes with 'toggle' and 'remove' methods.

  • Form Control

    A form using semantic html, required fields, and styled elements using CSS.

    I could learn more about responsive design, background images, and how to style form elements.

  • Karma Page

    Page made using HTML and CSS, responsive design, and semantic HTML.

    This page was especially challenging because I had to use flexbox and grid to create the layout. I improved my knowledge about CSS custom properties and how to use them.

    Besides that, this was the first project with more than one page. So, I had to learn more about style organization, using CSS in multiple pages, and creating new files for each specificity.
